# Arkanoid in Java

The classic Arkanoid game,  
written in Java using OOP principles.

![alt text](./readme_files/gameplay.gif 'Gameplay')

## Installation

1. Clone the repo:  
   `$ git clone https://github.com/sagyas/Arkanoid.git`
2. Compile:  
   `$ make compile`

## Usage

Run:  
`$ make run`
